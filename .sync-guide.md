# Package.json Sync Guide

## Quick Start

1. **Update `package.config.json`** with your package metadata (name, version, author, etc.)
2. **Run `pnpm sync`** to sync all package.json files

## What Gets Synced

### Root `package.json`
- name, version, description, author, license
- repository, bugs, homepage, keywords

### `package/package.json`
- name, version

### `web/package.json`
- name, version

## Usage

```bash
# Sync all package.json files
pnpm sync
```

## Optional: Auto-sync on Version Bump

If you want to automatically sync before version bumps, you can add this to your root `package.json`:

```json
{
  "scripts": {
    "preversion": "pnpm sync"
  }
}
```

This will automatically sync before running `npm version` commands.
