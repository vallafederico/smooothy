# Documentation

Add your package documentation here. This directory is for markdown documentation files that you can reference from your Astro site or include in your package.

## Suggested Structure

- `api.md` - API reference
- `getting-started.md` - Getting started guide
- `examples.md` - Usage examples
- `changelog.md` - Version changelog

You can link to these files from your Astro pages or include them in your package README.
